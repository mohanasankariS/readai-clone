# Read AI Clone - MVP Scaffold

Joins a Zoom / Google Meet / Microsoft Teams call as a bot, transcribes it,
and generates a summary + action items + highlights using Claude.

## Deploy
Push this repo to GitHub, connect it in Railway, set Root Directory to
`backend`, add RECALL_API_KEY, ANTHROPIC_API_KEY, and WEBHOOK_URL as
environment variables, then generate a public domain in Networking.
